# networkspeedcheckbot 

A **Telegram Bot** built in **Go**.  
Designed with **love**, for any telegram user to check internet speed.

## this bot depends speedtest-go and go-telegram-bot-api  modules
## Features

- Ping 
- Check Upload Speed
- Check Download Speed
